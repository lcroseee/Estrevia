```
Фаза 1: «Звёздный движок» ──── [мес. 1-3]   ── MVP
Фаза 2: «Созвездия»       ──── [мес. 4-9]   ── Рост
```

---

## Фаза 1: «Звёздный движок» (MVP) — месяцы 1-3

**Команда:** 1-2 fullstack + 1 дизайнер (частичная занятость)
**Бюджет:** $15,000-30,000 (или equity + свои силы)

### Что делаем

| Компонент | Описание |
|-----------|----------|
| Натальная карта | Sidereal + tropical toggle, SVG колесо, 3 ayanamsa |
| Планетарные часы | Real-time позиции, обновление каждую минуту |
| Лунный календарь | Фаза, знак, календарный вид на месяц |
| Эссе (120 шт.) | LLM-генерация + верификация, EN + ES |
| Лента данных | NASA DONKI + USGS Earthquake, real-time |
| Landing + waitlist | Красивая страница, сбор emails |
| PWA | Offline расчёт карт, installable |

### Результат

- Product Hunt запуск
- 5,000 регистраций
- Валидация product-market fit
- Первые метрики retention и conversion

## Фаза 2: «Созвездия» — месяцы 4-9

**Команда:** 3-5 разработчиков + 1 дизайнер
**Требуется:** доход от подписок или инвестиция

### Что делаем

| Компонент | Описание |
|-----------|----------|
| Совместимость | Синастрия 2 карт, аспекты, scoring |
| Транзиты | Наложение на натальную карту |
| Профили | Натальная карта в профиле, кастомизация |
| AI-аватары | Генерация по натальной карте (Stability AI) |
| Thoth Tarot | 78 оригинальных карт + интерпретации |
| Древо Жизни | 3 вида, маппинг на натальную карту |
| 2D карта системы | Солнечная система + зодиак, интерактивная |
| Календарь праздников | Равноденствия, солнцестояния, языческие праздники |
| Подписка Star | $9.99/мес, paywall для продвинутых фич |

### Результат

- 30,000 пользователей
- $10K+ MRR
- Премиум-подписка работает
- Данные для решения о Фазе 3

## Фаза 3: «Вселенная» — месяцы 10-18

**Команда:** 8-15 человек
**Требуется:** серьёзное финансирование (seed round)

### Что делаем

| Компонент | Описание |
|-----------|----------|
| 3D планеты | Three.js, NASA текстуры, zoom, звуки |
| Сообщения | Личные + группы (Stream SDK) |
| Музыкальный слой | Embed Spotify/YT, плейлисты на профиле |
| Дневник снов | Запись + AI-анализ по Юнгу |
| Аватар-помощник | Анимированный 2D-персонаж, озвучка |
| Mobile app | React Native (iOS + Android) |
| Виджеты | iOS/Android: лунная фаза, часы |
| Cosmos подписка | $19.99/мес, полный доступ |

### Результат

- 100,000+ пользователей
- $50K+ MRR
- Полноценная платформа
- Готовность к Series A

# Интернационализация (i18n)

---

## MVP: English only

Никакого locale routing, файлов переводов, next-intl.

### URL-структура

Без языкового префикса:

| Пример | URL |
|--------|-----|
| Карта | `estrevia.app/chart` |
| Эссе | `estrevia.app/essays/sun-in-pisces` |
| Лунный календарь | `estrevia.app/moon` |

### Контент

Все 120 эссе на английском. Одна версия каждого текста.

### UI strings

Захардкожены на английском. Никаких JSON-файлов переводов.

### Даты и числа

`Intl.DateTimeFormat` и `Intl.NumberFormat` с локалью `en-US`.

## Phase 2: план i18n

Когда аналитика покажет спрос из конкретных регионов (>10% трафика от ES/PT/RU-говорящих):

- `next-intl` или встроенный i18n Next.js
- URL-структура: `/es/chart`, `/pt/chart`
- 120 переводов эссе через Claude API + ручная проверка
- `hreflang` теги, `og:locale`
- Sitemap с language alternates

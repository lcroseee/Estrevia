# ESTREVIA

**Astrology · Astronomy · Community**

*El camino de las estrellas — Путь звёзд*

---

Платформа, где астрология встречается с реальной астрономией, а сообщество строится без алгоритмов и рекламы.

Estrevia объединяет профессиональную **сидерическую астрологию** с переключением на тропическую, **real-time данные NASA**, **эзотерическую традицию** (Thelema, Каббала, Tarot Thoth) и **социальную сеть** без алгоритмов — в одном приложении.

## Для кого

- **Астрологи** — первое красивое мобильное приложение для сидерической карты
- **Практикующие Thelema и Каббалу** — 777, Tree of Life, Thoth Tarot в одном интерфейсе
- **Astro-curious** — узнай свой настоящий знак, сравни с тропическим
- **Любители астрономии** — real-time солнечные вспышки, сейсмика, планетарные данные

## Возможности

**MVP (в разработке)**

- Натальная карта с переключателем sidereal / tropical (3 ayanamsa)
- Real-time планетарные часы
- Лунный календарь: фаза, знак, положение
- 120+ эссе по планетарным позициям (EN + ES)
- Лента данных NASA (солнечные вспышки) и USGS (землетрясения)
- PWA: работает офлайн, устанавливается на телефон

**Фаза 2** — синастрия, профили, AI-аватары, Thoth Tarot, Tree of Life, 3D солнечная система

**Фаза 3** — сообщения, каналы, музыка на профиле, мобильное приложение

## Стек

| Слой | Технология |
|------|-----------|
| Frontend | Next.js (App Router), TypeScript, Tailwind CSS, shadcn/ui |
| Астро-движок | Swiss Ephemeris (WASM) — расчёт на клиенте, офлайн |
| БД | PostgreSQL (Neon) + Prisma |
| Auth | Clerk |
| Платежи | Stripe |
| Хостинг | Vercel |
| AI | Claude API (контент), Stability AI (изображения) |
| Аналитика | PostHog |

## Принципы

- **Без рекламы.** Никогда. Монетизация через подписку.
- **Без алгоритма.** Хронологическая лента, никаких манипуляций.
- **Расчёт на клиенте.** Swiss Ephemeris WASM — мгновенный результат, офлайн, приватность.
- **Open source.** Код под AGPL-3.0. Контент (эссе) — proprietary.
- **Public domain контент.** Только легальные источники: 777, Equinox, NASA, Swiss Ephemeris.

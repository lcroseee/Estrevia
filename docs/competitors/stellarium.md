# Stellarium

> Золотой стандарт астрономической визуализации. Open source, 1.69B звёзд. Ноль астрологии.

---

## Обзор

| Параметр | Значение |
|----------|----------|
| **Тип** | Open source планетарий |
| **Версия** | Desktop v25.4, Mobile v1.15.2 (Feb 2026) |
| **Платформы** | Windows, macOS, Linux (бесплатно), iOS ($14), Android (бесплатно + Plus $14) |
| **Каталог** | 1.69 миллиарда звёзд (Gaia DR2), 2M+ deep sky объектов |
| **Лицензия** | GPL (desktop), проприетарный (mobile Plus) |

---

## Что делают хорошо ✅

- **Точность.** Научный уровень: real-time или историческое небо для любой даты/времени/локации
- **Масштаб данных.** 1.69B звёзд — невозможно конкурировать в объёме
- **Телескоп.** Контроль телескопов через WiFi/Bluetooth — уникально
- **Offline.** Полностью работает без интернета
- **Open source.** Desktop — GPL, огромное сообщество контрибуторов
- **Satellite tracking.** ISS и другие спутники в real-time

## Что делают плохо ❌

- **Ноль астрологии.** Никаких знаков, интерпретаций, карт, значений
- **Нет personal.** Нет «вашей» карты, нет привязки к дате рождения
- **Нет social.** Нет community, sharing, профилей
- **UI functional.** Для учёных, не для casual users
- **Mobile = платный.** iOS только платная версия ($14)

## Значение для Estrevia

Stellarium — **референс для 3D космоса** (Фаза 2-3), не конкурент.

**Что взять:**
- Качество рендеринга неба как цель
- NASA текстуры (public domain) — те же источники
- Offline philosophy для расчётов

**Где Estrevia другая:**
- Personal (натальная карта)
- Interpretive (значения, эссе)
- Esoteric (777, Tree of Life)
- Social (профили, community)

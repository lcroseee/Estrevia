# Платежи: Stripe

> ⚠️ Отсутствовал в изначальном стеке. Добавлен — подписки невозможны без платёжной системы.

## Что это простыми словами

Stripe принимает оплату: пользователь вводит карту → деньги приходят на твой счёт. Для подписок: Stripe автоматически списывает $9.99 каждый месяц, управляет отменами, trial-периодами, и отправляет чеки.

---

## Почему Stripe

| Причина | Для Estrevia |
|---------|-------------|
| **Стандарт** | 90%+ SaaS используют Stripe. Все туториалы, все примеры |
| **Subscriptions** | Stripe Billing: recurring payments, trials, upgrades, downgrades — из коробки |
| **Нет monthly fee** | Платишь только % с транзакций |
| **Checkout** | Hosted page: не нужно делать свою форму оплаты (PCI compliance) |
| **Next.js** | Отличные SDK и примеры для Next.js + Vercel |

### Комиссия

| Тип | Комиссия |
|-----|---------|
| Карта (US) | 2.9% + $0.30 |
| Карта (international) | 2.9% + $0.30 + 1.5% |
| Currency conversion | +1% |

### Реальная стоимость для наших тиеров

| Подписка | Gross | Stripe fee | Net | Effective rate |
|----------|-------|-----------|-----|---------------|
| Star ($9.99/мес) | $9.99 | $0.59 | $9.40 | 5.9% |
| Star ($79.99/год) | $79.99 | $2.62 | $77.37 | 3.3% |
| Cosmos ($19.99/мес) | $19.99 | $0.88 | $19.11 | 4.4% |

**Инсайт:** годовые подписки = лучший net margin (3.3% vs 5.9%). Ещё одна причина пушить annual plans.

## Альтернативы

### Lemon Squeezy

| Аспект | Подробности |
|--------|------------|
| **Комиссия** | 5% + $0.50 — дороже Stripe |
| **Плюс** | Merchant of Record: сами считают и платят налоги (VAT, sales tax) |
| **Куплен** | Stripe купил Lemon Squeezy (июль 2024). Работает независимо |
| **Минус** | Выплаты: net 15-30 дней (vs 2 дня у Stripe). Меньше контроля |
| **Для кого** | Solo-dev, кому лень разбираться с налогами |

### Paddle

| Аспект | Подробности |
|--------|------------|
| **Комиссия** | 5% + $0.50 — как Lemon Squeezy |
| **Плюс** | Merchant of Record. Лучший для глобальных SaaS с complex tax |
| **Минус** | Дороже. Меньше гибкости. Дольше setup |
| **Для кого** | SaaS с клиентами в 50+ странах |

### Решение для Estrevia

**Stripe.** Причины:
1. Самая низкая комиссия (2.9% + $0.30)
2. Аудитория US-first — не нужен global tax compliance на старте
3. Payouts за 2 дня (кешфлоу критичен для стартапа)
4. Лучшая интеграция с Next.js

**Если потребуется global tax:** подключить Stripe Tax ($0.50/транзакция) или мигрировать на Lemon Squeezy.

## Вердикт

**Stripe — единственный разумный выбор для MVP.** Нулевая monthly fee, лучшая комиссия, стандарт индустрии. Пушить годовые подписки для лучшего net margin.

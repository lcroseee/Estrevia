## Что делают хорошо ✅

- **Лунный фокус.** Moon phases, rituals, best days for activities — конкретная ценность
- **Wellness positioning.** Не «предсказания», а «живи в ритме луны»
- **Affordable.** $4/мес — ни��кий барьер
- **Dream analyzer.** Уникальная фича
- **Медитации.** Аудио-контент добавляет sticky value

## Что делают плохо ❌

- **Upsell-heavy.** Базовая подписка — entry point, потом каждая фича отдельно ($8-50)
- **Middling rating (4.0).** Не выдающийся
- **Мелко по астрологии.** Лунный календарь — это ~10% того, что может дать астрология
- **Нет сидерической.** Тропическая Луна vs сидерическая — разные знаки
- **Нет social.** Нет сообщества
- **Нет астрономических данных.** Только калькулятор фаз, не real-time NASA

## Прямое пересечение с Estrevia

Лунный календарь — **наша P0 фича**. Moonly = прямой конкурент в этом сегменте.

| Аспект | Moonly | Estrevia |
|--------|-------|---------|
| Лунный календарь | ✅ (core) | ✅ (одна из фич) |
| Lunar rituals | ✅ | Фаза 2 (эзотерические ритуалы) |
| Natal chart | Базовая | Полная (sidereal + tropical) |
| Dream journal | ✅ | Фаза 2 (Jung interpretation) |
| NASA данные | ✗ | ✅ (real-time) |
| Сидерическая Луна | ✗ | ✅ |
| 777 / эзотерика | ✗ | ✅ |

**Стратегия:** не конкурировать с Moonly на «wellness луне». Наш лунный календарь = точнее (sidereal) + глубже (эзотерический контекст) + научнее (NASA данные).

## Как читать файлы персон

Каждый файл содержит:

- **Кто они** — демография, психография, идентичность
- **Боли и проблемы** — что их раздражает, чего не хватает
- **Текущие инструменты** — чем пользуются сейчас и почему это не идеально
- **Где общаются** — конкретные площадки, каналы, сообщества
- **Триггер перехода** — что заставит их попробовать Estrevia
- **За что заплатят** — конкретные функции за paywall
- **Как привлечь** — каналы, messaging, контент
- **Риски сегмента** — что может пойти не так

## Стратегия привлечения по фазам

```
MVP (мес. 1-3):    Сидерические → Thelema → Тропические
                    ↓ органика: Reddit, Telegram, Facebook groups

Фаза 2 (мес. 4-9): + Gen Z → Астрономы
                    ↓ виральность: TikTok «твой настоящий знак»
                    ↓ синастрия: invite friends

Фаза 3 (мес. 10+): + Артисты / Музыканты
                    ↓ профили + музыка = MySpace reborn
                    ↓ word-of-mouth от креативного сообщества
```

# Nebula

> Агрессивная монетизация + плохая репутация. Анти-пример.

---

## Обзор

| Параметр | Значение |
|----------|----------|
| **Тип** | Astrology + psychic readings |
| **Платформы** | iOS, Android |
| **Downloads** | Миллионы |
| **Revenue** | ~$3.6M/год (est.) |
| **Рейтинг** | iOS: 4.6 (160K+ ratings), Trustpilot: плохой |
| **Цена** | $7.99/неделю (!), $24.99/мес, per-minute psychic chat |

---

## Что делают хорошо ✅

- **Широкий feature set.** Гороскопы + live psychic chat + лунный календарь + таро + совместимость
- **App Store рейтинг.** 4.6 с 160K+ отзывов (но likely inflated через prompt timing)
- **Revenue.** $8/н��делю x миллионы = серьёзные деньги

## Что делают плохо ❌

- **Агрессивный pricing.** $8/неделю = $32/мес — дороже Netflix. Непрозрачные условия
- **Сложная отмена подписки.** Массовые жалобы на Trustpilot: «не могу отменить», «неожиданные списания»
- **Per-minute psychic chat.** Модель exploitation — люди тратят $50-100 за сессию, не осознавая
- **Upsell hell.** Каждая функция — ещё один upsell
- **Нет глубины.** Генерический контент, нет expertise
- **Trustpilot reputation.** Массив негативных отзывов

---

## Урок для Estrevia

**Это анти-паттерн.** Всё, что Nebula делает — Estrevia не должна:

- ❌ Агрессивный pricing → ✅ Прозрачный freemium ($0 / $9.99 / $19.99)
- ❌ Per-minute charges → ✅ Flat subscription, всё включено
- ❌ Сложная отмена → ✅ Cancel в 1 клик (Settings)
- ❌ Upsell на каждом шагу → ✅ Free tier достаточно ценный сам по себе
- ❌ Fake reviews → ✅ Organic ratings через качество

**Nebula доказывает: даже плохой продукт в астрологии генерирует $3.6M/год. Качественный продукт может значительно больше.**

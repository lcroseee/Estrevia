# Карта функционала

Все функции разбиты на приоритеты:

- **P0** = должно быть в MVP
- **P1** = следующий релиз (Фаза 2)
- **P2** = важно, но не срочно
- **P3** = видение будущего

---

## 1. Астрологический движок

| Пр. | Функция | Сложность | Срок | Описание |
|-----|---------|-----------|------|----------|
| **P0** | Натальная карта | Средняя | 3-4 нед. | Сидерическая + тропическая, переключатель. Swiss Ephemeris WASM, Lahiri/Fagan-Bradley/Krishnamurti ayanamsa |
| **P0** | Планетарные часы | Средняя | 2-3 нед. | Real-time астрологическое колесо. Показывает текущие позиции планет в знаках. Обновляется каждую минуту |
| **P0** | Лунный календарь | Низкая | 1-2 нед. | Фаза Луны, знак, положение. Календарный вид на месяц. Уведомления о новолунии/полнолунии |
| **P0** | Почему сидерическая | Низкая | 1 нед. | Образовательная страница: разница между системами, визуальное сравнение, прецессия равноденствий |
| **P1** | Синастрия | Средняя | 2-3 нед. | Совместимость 2 карт. Аспекты между планетами, scoring, текстовые интерпретации |
| **P1** | Транзиты | Средняя | 2 нед. | Наложение текущих транзитов на натальную карту. Список активных аспектов |
| **P2** | Три-колесо | Высокая | 3-4 нед. | 3 карты одновременно (натал + транзит + прогрессия или натал + натал + транзит) |
| **P2** | Watch-виджет | Высокая | 3 нед. | Натальный календарь для Apple Watch. Требует React Native или native модуль |

## 2. Космический просмотрщик (3D)

| Пр. | Функция | Сложность | Срок | Описание |
|-----|---------|-----------|------|----------|
| **P1** | 2D карта системы | Средняя | 3-4 нед. | Солнечная система сверху + зодиакальный круг. Реальные позиции планет. Интерактивная |
| **P2** | 3D планеты | Высокая | 4-6 нед. | Вращение, NASA текстуры, освещение. Three.js + React Three Fiber |

| **P3** | Инфо о Земле | Очень высокая | 6-8 нед. | Погода, время, молнии, тектонические плиты в real-time на 3D глобусе |

## 3. Эзотерический контент

| Пр. | Функция | Сложность | Срок | Описание |
|-----|---------|-----------|------|----------|
| **P0** | Эссе по позициям | Низкая | 2-3 нед. | 120+ текстов (10 планет × 12 знаков). Генерация через Claude API + верификация астрологом. Основа: 777, Equinox (PD) |

| **P2** | Еврейский календарь | Средняя | 2 нед. | Лунно-солнечный календарь, праздники, каббалистические соответствия |

## 4. Социальная сеть

| Пр. | Функция | Сложность | Срок | Описание |
|-----|---------|-----------|------|----------|
| **P1** | Профиль | Средняя | 4 нед. | Натальная карта в профиле, базовая кастомизация (фон, цвета), планетарная «принадлежность» |
| **P1** | Посты / альбомы | Средняя | 3 нед. | Фото/текст посты, хронологическая лента (без алгоритма), комментарии |
| **P2** | AI-кастомизация | Высокая | 4-6 нед. | Генерация фона/темы профиля через промпты. «Сделай мой профиль в стиле nebula» |
| **P2** | Плейлист (MySpace) | Средняя | 2-3 нед. | Embed Spotify/YouTube плейлист на странице профиля. Автоплей при посещении |
| **P2** | Планетарный тег | Низкая | 1 нед. | Управляющая планета пользователя (по натальной карте) как badge |
| **P3** | Stories + Reels | Высокая | 6 нед. | Исчезающий контент + короткие видео |
| **P3** | Верификация | Низкая | 1-2 нед. | Синяя галочка для известных астрологов/авторов |

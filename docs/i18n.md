# Интернационализация (i18n)

---

## Стратегия

### MVP: 2 языка

| Язык | Код | Приоритет | Почему |
|------|-----|-----------|--------|
| English | `en` | Основной | Глобальный рынок, Reddit/ProductHunt аудитория |
| Español | `es` | Равный | Название продукта испанское, огромный рынок (500M+ носителей) |

### Фаза 2: расширение

| Язык | Код | Когда | Обоснование |
|------|-----|-------|-------------|
| Português | `pt` | При 10K+ юзеров из Бразилии | Бразилия = 2й рынок астрологии |
| Русский | `ru` | При запросе сообщества | Сильная эзотерическая традиция |
| Français | `fr` | При 50K+ юзеров | Франция = сильный рынок астрологии |

## Техническая реализация

### Подход: next-intl (или next built-in i18n)

```
app/
├── [locale]/           # Dynamic locale segment
│   ├── layout.tsx      # Locale provider
│   ├── page.tsx        # Home (localized)
│   ├── chart/
│   │   └── page.tsx
│   └── essays/
│       └── [slug]/
│           └── page.tsx
└── ...
```

### URL-структура

| Язык | URL |
|------|-----|
| English | estrevia.app/en/chart |
| Español | estrevia.app/es/chart |
| Default (en) | estrevia.app/chart → redirect to /en/chart |

### Файлы переводов

```
src/messages/
├── en.json          # UI strings
├── es.json          # UI strings
└── ...
```

### Что переводится

| Тип контента | Как переводится | Хранение |
|-------------|----------------|----------|
| UI strings (кнопки, навигация, labels) | Ручной перевод | JSON файлы |
| Эссе (120 шт.) | Claude API + ручная проверка | MDX файлы |
| Описания знаков/планет | Claude API + ручная проверка | MDX файлы |
| Лунный календарь | Claude API | MDX файлы |
| Юридические документы | Ручной перевод | Страницы |
| NASA data feed | Не переводится (eng + scientific terms) | API |
| Числа и градусы | Автоматически (Intl API) | — |

### Что НЕ переводится

- Астрологические глифы (♈♉♊...) — универсальны
- Названия планет в технических контекстах
- NASA/USGS данные (научный английский)
- Имена собственные (Crowley, Eshelman, Thoth)

## Форматы данных

| Данные | EN | ES | Как |
|--------|----|----|-----|
| Даты | Apr 4, 2026 | 4 abr. 2026 | `Intl.DateTimeFormat` |
| Время | 3:45 PM | 15:45 | `Intl.DateTimeFormat` + locale |
| Числа | 1,234.56 | 1.234,56 | `Intl.NumberFormat` |
| Градусы | 15°42'30" | 15°42'30" | Одинаково |
| Валюта | $9.99 | $9.99 | USD для всех (на старте) |
| Timezone | GMT-5 (EST) | GMT-5 (EST) | IANA + offset |

## Эзотерическая терминология

Некоторые термины не переводятся (сохраняют оригинал):

| Термин | EN | ES | Примечание |
|--------|----|----|-----------|
| Ayanamsa | Ayanamsa | Ayanamsa | Санскрит, универсально |
| Sephiroth | Sephiroth | Sephiroth | Иврит, универсально |
| Thelema | Thelema | Thelema | Греческий |
| Liber | Liber | Liber | Латинский |

Glossary файл в репозитории для consistency:

```
src/i18n/glossary.json
{
  "ayanamsa": { "en": "ayanamsa", "es": "ayanamsa" },
  "natal_chart": { "en": "natal chart", "es": "carta natal" },
  "sidereal": { "en": "sidereal", "es": "sideral" },
  "tropical": { "en": "tropical", "es": "tropical" },
  ...
}
```

## SEO и i18n

- `hreflang` теги на каждой странице
- `<link rel="alternate" hreflang="en" href="...">` / `hreflang="es"`
- `x-default` → English
- Sitemap с language alternates
- Open Graph: `og:locale` и `og:locale:alternate`

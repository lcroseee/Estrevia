# Jagannatha Hora

> Самый мощный бесплатный Jyotish-софт. Desktop, Windows, 2003 UI. Любим профессионалами.

---

## Обзор

| Параметр | Значение |
|----------|----------|
| **Тип** | Professional Vedic astrology desktop software |
| **Автор** | P.V.R. Narasimha Rao (IIT Madras, Rice University) |
| **Платформы** | Windows only (Linux через Snap) |
| **Цена** | Полностью бесплатно |
| **Эфемериды** | 12,899 BC — 16,900 AD |
| **Atlas** | 2.5M городов |

---

## Что делают хорошо ✅

- **Непревзойдённая глубина.** Все divisional charts, все dasha системы, Ashtakavarga, Tajaka, muhurta, KP, и многое другое
- **Бесплатно навсегда.** Volunteer проект
- **Полностью offline.** Не зависит от интернета
- **Доверие профессионалов.** Стандарт для серьёзных Jyotish практиков

## Что делают плохо ❌

- **Windows only.** Нет macOS, нет мобильного, нет web
- **UI из 2003.** Устаревший до невозможности
- **Steep learning curve.** Сотни параметров, нет onboarding
- **Только Jyotish.** Нет Western, нет тропической, нет эзотерики
- **Нет social.** Нет sharing, нет community

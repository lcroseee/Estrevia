# Email: Resend

## Что это простыми словами

Resend отправляет emails от имени твоего приложения: «Подтвердите email», «Ваша карта сохранена», «Новолуние сегодня — что это значит для вас». Не маркетинговые рассылки — а transactional emails, привязанные к действиям пользователя.

---

## Почему Resend

| Причина | Для Estrevia |
|---------|-------------|
| **DX** | API за 5 минут. React Email для красивых шаблонов |
| **Free tier** | 3,000 emails/мес бесплатно. Достаточно для MVP |
| **React Email** | Шаблоны пишутся на React — тот же язык, что и приложение |

---

## Что знать о Resend

### Подводные камни

- **Работает поверх Amazon SES.** Resend — не своя инфраструктура. Если SES упадёт — Resend упадёт
- **Data retention: 1 день (Free), 3 дня (Pro).** Сложно дебажить deliverability
- **Ноябрь 2025 — 3-часовой outage** из-за зависимости от Cloudflare
- **Блокировки аккаунтов.** Разработчики сообщают о внезапных блокировках после deliverability аудитов, с медленным саппортом

### Сравнение

| Сервис | Free tier | Надёжность | DX | Цена (10K/мес) |

## Стратегия

**MVP: Resend.** Бесплатно, быстро, React Email.

**Если email станет критичным** (waitlist nurture, daily readings, onboarding sequences): мигрировать на **Postmark**. Лучшая deliverability в индустрии (32ms median response, 0.03% error rate). Дороже, но email доходит.

---

## Вердикт

**Resend для старта, Postmark для масштаба.** На MVP Resend — правильный выбор. При 10K+ пользователей оценить Postmark для reliability.

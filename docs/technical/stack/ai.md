# AI: Claude API + Stability AI

## Что это простыми словами

Два AI-инструмента для разных задач:

- **Claude API** (Anthropic) — текстовый AI. Генерирует 240 эзотерических эссе, переводит EN↔ES, создаёт ежедневные readings, проверяет качество контента. Как нанять писателя-астролога, но за $5.
- **Stability AI** — графический AI. Генерирует аватары по натальной карте и оригинальные иллюстрации для Thoth-стиль Таро карт.

---

## Claude API

### Зачем

| Задача | Модель | Стоимость | Когда |
|--------|--------|-----------|-------|
| Генерация 120 эссе (EN) | Sonnet 4.6 | ~$2 | Разовая, до запуска |
| Перевод 120 эссе (ES) | Sonnet 4.6 | ~$2 | Разовая, до запуска |
| Fact-check (верификация) | Haiku 4.5 | ~$0.50 | Разовая |
| Ежедневные readings | Haiku 4.5 | ~$0.002/user/день | Runtime |
| AI-интерпретации (Фаза 2) | Sonnet 4.6 | ~$0.01/запрос | Runtime |

**Итого на генерацию контента:** ~$5. Это не опечатка.

### Почему Claude, а не GPT/Gemini

| Аспект | Claude | GPT-4o | Gemini |
|--------|--------|--------|--------|
| Качество длинного текста | ★★★ | ★★ | ★★ |
| Следование стилю (Crowley/эзотерика) | ★★★ | ★★ | ★★ |
| Цена (Haiku 4.5) | $0.25/M tokens in | $0.15/M (GPT-4o-mini) | $0.075/M (Flash) |
| Цена (Sonnet 4.6) | $3/M tokens in | $2.50/M (GPT-4o) | $1.25/M (Pro) |
| Безопасность контента | Строже | Средне | Средне |

**Ключ:** Claude лучше пишет длинные, стилистически выдержанные тексты. Для эзотерического контента со ссылками на 777 — это критично.

### Подводный камень: content policy

Claude может отказаться генерировать контент, связанный с «оккультизмом». 

**Митигация:** формулировать промпт как «историко-культурный анализ астрологических традиций», не как «магические инструкции». В практике работает.

## Stability AI

### Зачем

| Задача | Когда |
|--------|-------|
| AI-аватары по натальной карте | Фаза 2 |
| 78 оригинальных Таро иллюстраций (Thoth-стиль) | Фаза 2 |
| Фоны профилей | Фаза 2 |

### Почему Stability, а не DALL-E / Midjourney

| Аспект | Stability AI | DALL-E 3 | Midjourney |
|--------|-------------|---------|------------|
| API доступ | ✅ | ✅ (через OpenAI) | ❌ (только Discord) |
| Стиль контроль | Высокий (ControlNet) | Средний | Высокий |
| Цена | ~$0.02/image | ~$0.04/image | $10/мес flat |
| Self-host | ✅ (open weights) | ❌ | ❌ |
| Для Таро | Лучше (тонкий стиль) | OK | Лучше, но нет API |

**Для MVP:** Stability AI не нужен. Аватары и Таро — Фаза 2. На MVP — стандартные аватары.

## Вердикт

**Claude API для текста — без альтернатив** по качеству эзотерического контента. $5 на генерацию всего корпуса — стоимость ниже чашки кофе.

**Stability AI для графики — Фаза 2.** На MVP не нужен. Решение можно пересмотреть к тому моменту.

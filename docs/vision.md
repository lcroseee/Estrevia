# Видение продукта

## Что это

**ESTREVIA** (от исп. *estrella* + *vía* = путь звёзд) — веб-платформа (PWA), которая объединяет:

- **Профессиональную сидерическую астрологию** с переключением на тропическую систему
- **Real-time астрономические данные** (NASA, USGS, NOAA)
- **Эзотерическую традицию** (Thelema, Каббала, Tarot Thoth)
- **Социальную сеть** без алгоритмов, рекламы и манипуляций

В одном предложении:

> *Платформа, где астрология встречается с реальной астрономией, а сообщество строится без алгоритмов и рекламы.*

---

## Какую проблему решает

### 1. Поверхностность существующих приложений

Co-Star, The Pattern и другие используют **только тропическую систему**. Ни одно приложение не даёт сидерическую карту + эзотерический контекст (Древо Жизни, Thoth) в одном интерфейсе.

**Для кого это проблема:** практикующие астрологи, изучающие ведическую астрологию, последователи Thelema и Каббалы.

### 2. Разрыв между астрологией и астрономией

Ни одно приложение не объединяет **real-time научные данные** (солнечные вспышки, тектоника, погода планет) с астрологической интерпретацией. Астрология и астрономия живут в разных мирах.

**Для кого это проблема:** все, кто хочет видеть астрологию не как «гороскоп в газете», а как систему, связанную с реальными космическими процессами.

### 3. Токсичность соцсетей

Люди устали от алгоритмов, рекламы и манипуляций. Растёт спрос на «чистые» платформы (BeReal, Mastodon, Bluesky).

**Для кого это проблема:** все пользователи соцсетей, особенно Gen Z.

### Конкурентный ландшафт

| Продукт | Сидерич. | Соц.сеть | 3D космос | Эзотерика | Анти-алгоритм |
|---------|----------|----------|-----------|-----------|---------------|

| The Pattern | ✗ | ✓ | ✗ | ✗ | ✗ |

| Stellarium | ✗ | ✗ | ✓ | ✗ | N/A |
| **ESTREVIA** | **✓** | **✓** | **✓** | **✓** | **✓** |

**Ключевой инсайт:** ни один конкурент не закрывает более 2 из 5 категорий.

### Что НЕ будет делать Estrevia

- **Не будет показывать рекламу.** Никогда. Монетизация через подписку.
- **Не будет алгоритмической ленты.** Хронологическая лента + подписки.
- **Не будет «попсовой» астрологией.** Глубина, а не «что ждёт Овна на этой неделе».
- **Не будет хостить чужую музыку.** Embed Spotify/YouTube, не своё CDN.
- **Не будет медицинскими/финансовыми советами.** Disclaimer во всех текстах.
- **Не будет использовать чужой копирайт.** Только public domain + собственный контент.

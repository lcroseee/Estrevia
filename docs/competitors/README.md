## Пробелы на рынке (Market Gaps)

### Gap 1: Сидерическая + тропическая в одном красивом приложении

> **Ни одно Western app не предлагает сидерическую карту как primary feature.**

- Co-Star, CHANI, Pattern, Nebula, Sanctuary, Moonly, TimePassages — **все только тропические**
- Astro.com имеет сидерическую, но спрятана в настройках (5 кликов)
- AstroSage и Jagannatha Hora — только Jyotish, без Western astrology
- Time Nomad (iOS) поддерживает обе системы, но нишевый и малоизвестный
- **Никто не делает side-by-side toggle sidereal/tropical**

**Возможность Estrevia:** первое красивое мобильное приложение с sidereal/tropical toggle

### Gap 2: Астрология + Real-time астрономия

> **Ноль приложений объединяют real-time NASA данные с астрологической интерпретацией.**

- Co-Star использует JPL эфемериды для расчётов, но не показывает real-time космические события
- Stellarium, Star Walk, SkySafari — чистая астрономия, ноль астрологии
- SpaceWeather.com — данные есть, но UI из 1999 и нет астрологии
- **Мост между астрономией и астрологией не существует**

**Возможность Estrevia:** единственная платформа, где солнечные вспышки + землетрясения + погода планет + астрологическая карта

### Gap 3: Эзотерическая традиция (Thelema, Каббала, 777)

> **Территория полностью не освоена. Ноль серьёзных конкурентов.**

- Thelema 93 — 32 рейтинга, крошечный, только утилиты (планетарные часы, Resh)
- Kabbalah App — нумерология, не Hermetic Qabalah
- Open 777 — open-source проект одного человека на Netlify, не приложение
- **Ни одно приложение не интегрирует 777 соответствия + натальную карту + Tree of Life**
- **Ни одно приложение не предлагает digital Thoth Tarot с глубокими интерпретациями**

**Возможность Estrevia:** первая digital платформа для серьёзных практикующих Thelema/Qabalah/Thoth

### Gap 4: Анти-алгоритмическая духовная соцсеть

> **Не существует.**

- FolkRealms, Coven Cloud, Pagaro — только для язычников, кро��ечные
- Soulify — 2/5 звёзд, плохая модерация
- Mastodon/Bluesky — general purpose, не духовные
- **Ни одна платформа не совмещает астрологию + соцсеть + без алгоритма**

**Возможность Estrevia:** первая астрологическая платформа без алгоритмов, с фокусом на контент и сообщество (Phase 2+)

### Gap 5: Красивый профессиональный инструмент

> **Gap между Co-Star (красиво, но попсово) и Astro.com (мощно, но уродливо).**

- Co-Star: 4/10 по глубине, 9/10 по дизайну
- Astro.com: 10/10 по глубине, 3/10 по дизайну
- **Никто не совмещает оба**

**Возможность Estrevia:** красота Co-Star + глубина Astro.com

# Карта функционала

Все функции разбиты на приоритеты:

- **P0** = должно быть в MVP
- **P1** = следующий релиз (Фаза 2)
- **P2** = важно, но не срочно
- **P3** = видение будущего

---

## 1. Астрологический движок

| Пр. | Функция | Сложность | Срок | Описание |
|-----|---------|-----------|------|----------|
| **P0** | Натальная карта | Средняя | 3-4 нед. | Сидерическая + тропическая, переключатель. Swiss Ephemeris (серверный API), Lahiri ayanamsa (MVP). Fagan-Bradley/Krishnamurti — Фаза 2 |
| **P0** | Лунный календарь | Низкая | 1-2 нед. | Фаза Луны, знак, положение. Календарный вид на месяц |
| **P0** | «Космический паспорт» — viral share | Средняя | 1-2 нед. | Виральная карточка-идентичность: Солнце/Луна/Асцендент + управляющая планета + элемент + "1 из X%" (редкость комбинации). OG image (`@vercel/og`), Web Share API с pre-filled текстом (EN), deep link `/s/[id]`, download PNG. Основной growth loop продукта |
| **P0** | Почему сидерическая | Низкая | 1 нед. | Образовательная страница: разница между системами, визуальное сравнение, прецессия равноденствий |
| **P0** | Планетарные часы | Средняя | 2-3 нед. | Real-time планетарные часы. PlanetaryHourBar — основной механизм ежедневного удержания (показывает текущий планетарный час в header). Без этого retention падает до "прочитал эссе → ушёл" |
| **P1** | MCP-сервер | Низкая | 3-5 дней | Обёртка над `/api/chart/calculate` для AI-ассистентов. Публикация в Smithery. Реально 3-5 дней с тестированием, не 1 день. После стабилизации API |
| **P1** | Лунные карточки | Низкая | 1 нед. | Персонализированная share card на каждое новолуние/полнолуние: "Полнолуние в Скорпионе — что это значит для Рыб?" Push-уведомление + готовая карточка для шеринга. 2× в месяц — регулярный повод для распространения |
| **P1** | Birthday Wrapped | Средняя | 2 нед. | Годовой астро-итог на день рождения пользователя: ключевые транзиты года, управляющие влияния, персональная статистика. Серия из 5-7 карточек (как Spotify Wrapped). Требует транзитов |
| **P1** | Синастрия | Средняя | 2-3 нед. | Совместимость 2 карт. Аспекты между планетами, scoring, текстовые интерпретации |
| **P1** | Транзиты | Средняя | 2 нед. | Наложение текущих транзитов на натальную карту. Список активных аспектов |
| **P2** | Три-колесо | Высокая | 3-4 нед. | 3 карты одновременно (натал + транзит + прогрессия или натал + натал + транзит) |
| **P2** | Watch-виджет | Высокая | 3 нед. | Натальный календарь для Apple Watch. Требует React Native или native модуль |

## 4. Социальный слой

> Estrevia — астрологическая платформа с виральным шерингом, не социальная сеть. Социальные функции — Фаза 2+.

| Пр. | Функция | Сложность | Срок | Описание |
|-----|---------|-----------|------|----------|
| **P2** | Профиль | Средняя | 4 нед. | Натальная карта в профиле, базовая кастомизация (фон, цвета), планетарная «принадлежность» |
| **P2** | Посты / альбомы | Средняя | 3 нед. | Фото/текст посты, хронологическая лента (без алгоритма), комментарии |
| **P2** | AI-кастомизация | Высокая | 4-6 нед. | Генерация фона/темы профиля через промпты. «Сделай мой профиль в стиле nebula» |
| **P2** | Плейлист (MySpace) | Средняя | 2-3 нед. | Embed Spotify/YouTube плейлист на странице профиля. Автоплей при посещении |
| **P2** | Планетарный тег | Низкая | 1 нед. | Управляющая планета пользователя (по натальной карте) как badge |
| **P3** | Stories + Reels | Высокая | 6 нед. | Исчезающий контент + короткие видео |
| **P3** | Верификация | Низкая | 1-2 нед. | Синяя галочка для известных астрологов/авторов |

## Сводка по фазам

| Фаза | P-уровень | Функций | Общая сложность |
|------|-----------|---------|----------------|
| **MVP** (мес. 1-3) | P0 core | 4 (карта, луна, паспорт, «почему сидерическая») | Средняя |
| **MVP+** (мес. 3-4) | P0 optional | 2 (план. часы, MCP) + Stripe + PWA polish | Средняя |
| **Фаза 2** (мес. 4-9) | P1 | 11 | Высокая |
| **Фаза 2+** (мес. 6-12) | P2 | 15 | Очень высокая |
| **Фаза 3** (мес. 10-18) | P3 | 14 | Экстремальная |

> **Реалистичная оценка MVP:** 4 core-функции + инфраструктура + контент + landing = **14-18 недель** для solo-разработчика. Минимальный запуск (карта + эссе + landing + share) возможен за **10-12 недель** если отложить Stripe, PWA polish и планетарные часы.

# Дизайн и UX

## Дизайн-философия

### «Космический минимализм»

Тёмная тема по умолчанию (cosmos feel), но с чистой типографикой и пространством. Не перегруженный, не «магический» — скорее как **Linear встречает Stellarium**.

| Принцип | Описание |
|---------|----------|
| **Dark by default** | Глубокий тёмный фон (#0A0A0F — почти чёрный с синим оттенком), не чистый #000 |
| **Типографика** | Чистый sans-serif (Inter/Geist) для UI, serif (Crimson Pro/Playfair) для эзотерических текстов |
| **Пространство** | Много воздуха. Контент дышит. Нет перегрузки элементами |
| **Цвет как акцент** | Основной UI — монохром. Цвет только для планет, знаков, акцентов |
| **Анимации** | Плавные, subtle. Framer Motion. Не flashy, а felt |

### Progressive Disclosure

Сложность открывается постепенно:

- **Новичок:** видит простую карту, знак Солнца, Луны, восходящий + короткие описания
- **Продвинутый:** включает градусы, минуты, аспекты, дома
- **Эксперт:** полный расклад с орбисами, деканатами, фиксированными звёздами

Переключение через настройки + автоматический detection (если пользователь не взаимодействует с продвинутыми элементами — не показывать их).

### Mobile First

**80%+ пользователей** астро-приложений на мобильных. Дизайн начинается с 375px (iPhone SE), адаптируется вверх.

- Тач-жесты для колеса натальной карты (вращение, zoom)
- Навигация: нижний tab bar (5 вкладок максимум)
- Карточки контента вместо длинных страниц

### Навигационный принцип

**Колесо натальной карты — центральный hub.** Всё доступно через тапы по планетам/знакам на колесе:

```
       Тап на планету → эссе по позиции
       Тап на знак → описание знака
       Тап на дом → интерпретация дома
       Тап на аспект → описание аспекта
       Свайп → переключение sidereal/tropical
       Long press → детали (градусы, орбис)
```

## Дизайн-система (план)

### Компоненты (shadcn/ui как основа)

MVP использует shadcn/ui с кастомной тёмной темой:

| Компонент | shadcn/ui | Кастомизация |
|-----------|-----------|-------------|
| Button | ✅ | Планетарные цвета для CTA |
| Card | ✅ | Полупрозрачный фон (glass effect) |
| Dialog/Modal | ✅ | Тёмный backdrop с blur |
| Tabs | ✅ | Sidereal/Tropical toggle |
| Calendar | ✅ | Лунные фазы как маркеры |
| Tooltip | ✅ | Астрологические подсказки |
| Navigation | ✅ | Нижний tab bar (mobile) |
| **Chart Wheel** | ❌ Custom | SVG, интерактивное, touch-ready |
| **Planet Icon** | ❌ Custom | Глиф + цвет + анимация |
| **Moon Phase** | ❌ Custom | SVG с тенью, real-time |

### Типографика

| Роль | Шрифт | Вес | Размер |
|------|-------|-----|--------|
| UI / навигация | Geist Sans | 400, 500, 600 | 14-16px |
| Заголовки | Geist Sans | 600, 700 | 20-48px |
| Эзотерический контент | Crimson Pro | 400, 400i | 18-20px |
| Числа / градусы | Geist Mono | 400 | 14px |

### Иконки

- **UI иконки:** Lucide React (совместим с shadcn/ui)
- **Астрологические глифы:** кастомный SVG набор (♈♉♊♋♌♍♎♏♐♑♒♓ + планеты)
- **Лунные фазы:** кастомный SVG (8 основных фаз)

## Адаптивность

| Брейкпоинт | Устройство | Layout |
|-----------|-----------|--------|
| < 640px | Мобильный | Одна колонка, bottom tabs, карточки |
| 640-1024px | Планшет | 2 колонки, sidebar visible |
| > 1024px | Десктоп | 3 колонки, полное колесо, sidebar |

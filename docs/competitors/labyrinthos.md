# Labyrinthos Academy

> Лучшее таро-приложение. «Duolingo for Tarot». Доказал, что gamified esoteric education работает.

---

## Обзор

| Параметр | Значение |
|----------|----------|
| **Тип** | Tarot/divination learning + reading app |
| **Founder** | Tina Gong (designer + developer) |
| **Платформы** | iOS, Android |
| **Reviews** | 36K+ positive (Google Play) |
| **Цена** | Free + $0.10/AI reading, $9.99 premium |
| **Охват** | Tarot, Lenormand, Runes, Astrology basics |

---

## Что делают хорошо ✅

- **Gamified learning.** «Duolingo for Tarot» — bite-size уроки, прогресс, quiz
- **Design.** Минималистичный, gold-on-black. Работа Tina Gong — узнаваемый стиль
- **100% ad-free.** Монетизация через optional purchases + physical decks
- **«Your Mirror».** Анализирует все readings и находит повторяющиеся паттерны — умная фича
- **Мультиформатный.** Таро + Lenormand + Руны + Астрология — одно приложение
- **Physical + Digital.** Golden Thread дек ($55) + app — holistic business model

## Что делают плохо ❌

- **Template-based readings.** Каждая карта = одна и та же интерпретация. Нет контекстной AI-генерации
- **Нет Thoth.** Только Rider-Waite и собственные деки
- **Нет эзотерической глубины.** 777, Qabalah, Tree of Life — отсутствуют
- **Нет community.** Нет social, нет discussion, нет sharing
- **Астрология — поверхностная.** Basics only, нет полной карты

---

## Урок для Estrevia

Labyrinthos **доказала рынок**: gamified esoteric education = business ($55 deck + app monetization + 36K reviews).

**Что перенять:**
- Gamification для обучения эзотерике (прогресс по Tree of Life, quiz по 777)
- Physical + digital model (Estrevia branded Thoth deck в будущем)
- Ad-free = trust

**Где Estrevia идёт дальше:**
- Thoth (не Rider-Waite)
- 777 correspondences, не просто card meanings
- Астрология как core, не дополнение
- Natальная карта → Tree of Life → Tarot в одном flow

# Offline Architecture: что работает без интернета

---

## Простыми словами

Estrevia как PWA может работать без интернета. Но не всё — только то, что не требует сервера. Расчёт карты = офлайн (WASM на устройстве). Сохранение карты = нужен интернет (сервер + БД).

---

## Service Worker: стратегии кэширования

```
┌─────────────────────────────────────────────────────────┐
│                   Service Worker                        │
│                                                         │
│  Стратегия 1: CACHE FIRST (статика)                     │
│  ─────────────────────────────────                      │
│  Файл запрошен → есть в кэше? → ДА → отдать из кэша   │
│                                  НЕТ → fetch → кэш     │
│                                                         │
│  Применяется к:                                         │
│    ├── WASM (Swiss Ephemeris)                           │
│    ├── Шрифты (.woff2)                                  │
│    ├── Иконки, глифы                                    │
│    ├── App shell (JS/CSS bundles)                       │
│    └── Статичные изображения                            │
│                                                         │
│                                                         │
│  Стратегия 2: STALE WHILE REVALIDATE (контент)          │
│  ──────────────────────────────────────────              │
│  Файл запрошен → отдать из кэша (мгновенно)            │
│                → фоном: fetch свежую версию → обновить  │
│                                                         │
│  Применяется к:                                         │
│    ├── Эссе (MDX → HTML)                               │
│    ├── Лунный календарь                                 │
│    └── Описания знаков/планет                           │
│                                                         │
│                                                         │
│  Стратегия 3: NETWORK FIRST (динамика)                  │
│  ─────────────────────────────────────                   │
│  Файл запрошен → fetch с сервера                        │
│                  ОК → отдать + кэш                      │
│                  FAIL → отдать из кэша (stale)          │
│                                                         │
│  Применяется к:                                         │
│    ├── /api/feed (NASA/USGS)                            │
│    ├── /api/chart/list (список карт)                    │
│    └── Auth status                                      │
│                                                         │
│                                                         │
│  Стратегия 4: NETWORK ONLY (нет кэша)                   │
│  ────────────────────────────────────                    │
│  Применяется к:                                         │
│    ├── /api/chart/save (POST)                           │
│    ├── /api/stripe/* (POST)                             │
│    └── /api/webhooks/* (POST)                           │
└─────────────────────────────────────────────────────────┘
```

## Offline UI

```
Нормальный режим (online):
┌──────────────────────────────────────┐
│  ☀ Карта  │  ☽ Луна  │  📖 Эссе   │  ← обычный UI
└──────────────────────────────────────┘

Офлайн режим:
┌──────────────────────────────────────┐
│  ⚡ Offline                          │  ← тонкий баннер вверху
├──────────────────────────────────────┤
│                                      │
│  ☀ Карта (работает нормально)       │
│  ☽ Луна (работает нормально)        │
│  📖 Эссе (кэшированные — ОК,       │
│           новые — "Нужен интернет") │
│  🌍 Данные — "Последнее обновление: │
│              2 часа назад"           │
│  💾 Сохранить — "Сохранится при     │
│                  подключении"        │
│                                      │
└──────────────────────────────────────┘

Восстановление:
┌──────────────────────────────────────┐
│  🟡 Подключение...                   │  ← кратко
├──────────────────────────────────────┤
│  → Background sync:                  │
│    Обновить NASA/USGS данные         │
│    Синхронизировать pending saves     │
│    Обновить stale кэш                │
└──────────────────────────────────────┘
```

## PWA Install Flow

```
Визит 1: Пользователь открывает estrevia.app
         Service Worker регистрируется
         WASM начинает кэшироваться в фоне
         (пользователь не знает)

Визит 2: Всё уже в кэше → мгновенная загрузка
         Chrome показывает "Add to Home Screen" (если criteria met)
         Мы показываем кастомный prompt ПОСЛЕ расчёта карты:
         
         ┌─────────────────────────────────┐
         │  ☆ Установить Estrevia?         │
         │                                 │
         │  • Мгновенный доступ            │
         │  • Расчёт карт офлайн           │
         │  • Лунный календарь всегда      │
         │                                 │
         │  [Установить]    [Не сейчас]    │
         └─────────────────────────────────┘

Install:  Иконка на домашнем экране
          Standalone window (без browser chrome)
          Splash screen (ESTREVIA logo, #0A0A0F bg)
```

## Объём кэша по компонентам

| Компонент | Размер | Стратегия |
|-----------|--------|-----------|
| App shell (HTML/CSS/JS) | ~3MB | Cache First |
| Swiss Ephemeris WASM | ~2MB | Cache First |
| Эссе (120 × ~3KB) | ~360KB | Stale While Revalidate |
| Описания (34 × ~2KB) | ~68KB | Stale While Revalidate |
| Шрифты (Geist + Crimson) | ~500KB | Cache First |
| Иконки / глифы | ~200KB | Cache First |
| Города (autocomplete) | ~500KB | Cache First |
| Timezone data | ~100KB | Cache First |
| Лунный календарь | ~50KB | Stale While Revalidate |
| NASA/USGS (последние) | ~20KB | Network First |
| **Итого** | **~7MB** | |

При использовании (читаешь эссе → оно кэшируется) объём растёт до ~11MB максимум.

# GTM Стратегия и маркетинг

## Принцип: paid-first валидация

> **Органика на старте — миф.** У нового проекта с нулевым брендом, нулевой аудиторией и нулевым трафиком ждать органику — значит ждать 3-6 месяцев без данных. Платная реклама даёт ответ за 7-10 дней. Реклама — это не трата, а покупка информации.

#### Основной growth loop

«Космический паспорт» — viral share card. Пользователь делает наш маркетинг, потому что хвастается собой (Солнце/Луна/Асцендент + "1 из 8%"), а не Estrevia. Каждый share = бесплатный показ для exact target audience. Web Share API с pre-filled текстом (EN). Цель: viral coefficient > 1.0.

**Повторяемость шеринга (Фаза 2):**
- Лунные карточки: 2×/мес (новолуние/полнолуние) — персонализированная карточка
- Birthday Wrapped: 1×/год — годовой астро-итог, FOMO как у Spotify

## Meta Ads: стратегия

### Принцип: реклама с первого дня

> **Meta Ads — инструмент валидации, а не только масштабирования.** $100-200 на старте покупает данные (какой креатив работает, кто конвертится, где воронка ломается), которые органика даст через 3-6 месяцев, а может не дать никогда.

### Технические требования (ДО первого доллара в рекламу)

| Что | Почему обязательно |
|-----|-------------------|
| **Meta Pixel** | Код в `<head>` сайта. Отслеживает поведение на сайте |
| **Conversions API** | Серверные события. Без CAPI теряем 30-40% данных из-за блокировщиков и iOS. Next.js API routes → `fetch` к Meta Graph API |
| **PostHog с событиями воронки** | Meta говорит «кого искать». PostHog говорит «что чинить» |
| **UTM-разметка** | Чтобы отличить рекламный трафик от органики |
| **Отдельный лендинг для рекламы** | Не главная страница. Одно действие, ноль отвлечений: заголовок → ввод даты → результат |

> **Подробности о Conversions API — см. `docs/analytics.md`, раздел «Conversions API (CAPI)».**

### Структура кампании (стартовый тест — $200)

```
1 кампания, CBO, $20/день, 10 дней

1 адсет: Broad (женщины 22-38, без интересов)
  — НЕ дробить на 3 адсета по $6.67. Meta в 2026 находит
    аудиторию лучше ручного таргетинга. Пусть работает.

Оптимизация: Landing Page View (все 10 дней)
  — НЕ переключать на chart_calculated на день 4.
    Для переключения нужно 50 событий/неделю — не наберём.
    chart_calculated отслеживаем в PostHog, но Meta
    оптимизируем на дешёвое событие.

3 креатива:
├── Reels 9:16: «Ты не Овен. Вот доказательство.» (15 сек)
├── Reels 9:16: Запись экрана — ввожу дату → «подожди, что?!» (15 сек)
└── Статика: Космический Паспорт + «Узнай свой настоящий знак»
```

**День 3:** креатив с CTR < 1% — выключить. Адсет с CPC > $3 — выключить. Не трогать то, что работает.

### Ретаргетинг — главная ценность малого бюджета

Собираем пиксельную аудиторию из **всех** каналов (TikTok, инфлюенсеры, Reddit, MCP). За $3-5/день показываем рекламу тем, кто уже рассчитал карту, но не зарегистрировался:

- Аудитория маленькая → бюджет маленький → работает
- CTR и конверсия в 5-10× выше холодного трафика
- $50-100/мес на ретаргетинг может дать больше подписок, чем $400 на холодный трафик

### Распределение бюджета $500

| Канал | Бюджет | Когда |
|-------|--------|-------|
| Meta: валидация воронки (холодный трафик) | $100 | Неделя 9 (день запуска +5) |
| Meta: масштабирование winning ad | $150 | Неделя 11-12 |
| Meta: ретаргетинг | $50 | Неделя 11-12 |
| TikTok Spark Ads | $100 | Неделя 10-12 |
| Инфлюенсеры | $100 | Неделя 9 + 11 |
| **Итого** | **$500** | |

### Масштабирование (после валидации)

| Этап MRR | Бюджет Meta/мес | Стратегия |
|----------|----------------|-----------|
| $0-2K | $200-500 | Winning креатив + ретаргетинг |
| $2-10K | $1,500-3,000 | + lookalike на `chart_calculated` и платящих |
| $10-30K | $3,000-5,000 | + TikTok Ads, Telegram Ads |
| $30K+ | $5,000+ | Полноценное масштабирование, агентство или инхаус-таргетолог |

### Воронка

```
Meta Ad → Отдельный лендинг «Узнай свой настоящий знак»
  → Расчёт карты БЕЗ регистрации (снижаем friction)
    → Результат + Космический Паспорт
      → Шеринг (viral loop)
      → «Полный разбор?» → Регистрация
        → Email nurture (Resend: 3-5 писем за 7 дней)
          → Soft paywall → Подписка
```

### Что измеряем на тестовые $200

> На $200 подписок будет 5-15 — статистически ничтожно. Измеряем не подписки, а сигналы спроса:

| Метрика | Порог «ок» | Порог «отлично» |
|---------|-----------|----------------|
| CPC (стоимость клика) | < $2 | < $1 |
| Клик → расчёт карты | > 40% | > 60% |
| Расчёт → регистрация | > 15% | > 25% |
| Расчёт → шер Паспорта | > 10% | > 25% |
| CTR winning креатива | > 1.5% | > 3% |

### Сценарии после теста

| Результат | Действие с оставшимися $300 |
|-----------|---------------------------|
| **CPC < $1.50, конверсия в расчёт > 40%** | Масштабируем winning ad + ретаргетинг. Строим MVP уверенно |
| **CPC $1.50-3, конверсия 20-40%** | Тестируем новые креативы/лендинг на $150. $150 на запуск |
| **CPC > $3 или конверсия < 15%** | Стоп. $300 сохранены. Проблема в подаче или продукте — разбираемся |

### Космический Паспорт — техническая схема

```
1. Пользователь рассчитал карту
2. Нажал «Поделиться»
3. Генерируется страница /s/[id]
4. OG-картинка: /api/og/passport/[id] (Satori, @vercel/og)
   — превью в Telegram, iMessage, Twitter, WhatsApp
5. Друг видит превью → кликает → /s/[id]
6. Страница: Паспорт друга + кнопка «Узнай свой знак»
7. Друг считает → свой Паспорт → шарит → цикл повторяется
```

**Шеринг:**
- Мобилка: `navigator.share()` — нативная системная шторка
- Десктоп: кнопки копирования ссылки, Telegram URL, Twitter intent
- Скачать PNG для Instagram Stories
- Pre-filled текст EN

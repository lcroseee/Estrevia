# DNS и домены: Cloudflare

## Что это простыми словами

DNS — это «телефонная книга интернета». Когда ты набираешь estrevia.app → DNS переводит это в IP-адрес сервера Vercel. Cloudflare делает это быстрее всех в мире (1.1.1.1 — самый быстрый публичный DNS-резолвер).

Кроме DNS, Cloudflare даёт: DDoS-защиту (бесплатно), SSL-сертификаты (бесплатно), и регистрацию доменов по себестоимости (без наценки).

---

## Почему Cloudflare

| Причина | Подробности |
|---------|------------|
| **Самый быстрый DNS** | Не маркетинг — объективно. Медиана ответа ~11ms |
| **Домены по себестоимости** | .com ~$10.11/год. .app — wholesale цена. Без наценки |
| **DDoS protection** | Бесплатно на всех тарифах. Для web-app — критично |
| **$0** | DNS, DDoS, SSL — бесплатно навсегда |

---

## ⚠️ Важно: Cloudflare + Vercel

**Не включать Cloudflare proxy (оранжевое облако)** при использовании Vercel. Причина: Vercel Bot Protection не работает корректно через Cloudflare proxy. Trафик видится как «от Cloudflare», а не от реального пользователя.

**Правильная настройка:**
1. Регистрация домена на Cloudflare (дешевле всех)
2. DNS records указывают на Vercel (серое облако = DNS only, без proxy)
3. SSL/TLS обеспечивает Vercel (автоматически)

---

## Альтернативы

| Сервис | Плюсы | Минусы |
|--------|-------|--------|
| **Vercel Domains** | Без наценки. Нативная интеграция | DNS не такой быстрый. Нет DDoS protection уровня Cloudflare |
| **Namecheap** | Привычно, дёшево | DNS медленнее. Нет DDoS |
| **Google Domains** | Был хорош, закрылся (перешёл в Squarespace) | RIP |

---

## Вердикт

**Cloudflare — бесплатно, быстрее всех, надёжнее всех.** Единственная настройка: серое облако (DNS only) для Vercel. Зарегистрировать estrevia.app + estrevia.com здесь.
